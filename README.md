# 🚀 Cyber Cafe Management System

> Desktop application for managing an internet cafe, built with Python and Tkinter.

---

## 📌 Overview

This project is a **cyber cafe management system** that helps manage users, machine usage time, and food & drink ordering.

It is designed as a desktop application with a graphical user interface (GUI), focusing on real-world management features.

---

## ✨ Features

### 👤 User Management

* Login / Logout system
* Create and manage user accounts
* Change password
* Recharge balance

### 💻 Machine Management

* Track machine usage time
* Manage active machines
* Monitor online/offline status

### 🍔 Food & Drink Ordering

* Display menu with images
* Add items to cart
* Calculate total cost
* Handle user orders

### 🛠️ Admin Features

* Admin account management
* System monitoring
* Data management via JSON

---

## 🧠 Technologies Used

* **Python** – Core programming language
* **Tkinter** – GUI development
* **JSON** – Data storage
* **OOP (Object-Oriented Programming)**
